Project Structure Overview
Your project is a React + TypeScript + Tailwind CSS application. Here are all the key files:

1. Main App Component (client/src/App.tsx)
import { Switch, Route } from "wouter";
import { queryClient } from "./lib/queryClient";
import { QueryClientProvider } from "@tanstack/react-query";
import { Toaster } from "@/components/ui/toaster";
import { TooltipProvider } from "@/components/ui/tooltip";
import NotFound from "@/pages/not-found";
import Home from "@/pages/home";
function Router() {
  return (
    <Switch>
      <Route path="/" component={Home} />
      <Route component={NotFound} />
    </Switch>
  );
}
function App() {
  return (
    <QueryClientProvider client={queryClient}>
      <TooltipProvider>
        <Toaster />
        <Router />
      </TooltipProvider>
    </QueryClientProvider>
  );
}
export default App;

To Use This Code on Your Website:
Option 1: Copy Just the Component

Copy the home.tsx component and use it directly in your React app
Install dependencies: npm install framer-motion
Import and use: <BrokenTitle />
Option 2: Set Up the Entire Project

Clone/download this repo
Run npm install
Run npm run dev:client to start the dev server
Access at http://localhost:5000
Option 3: Extract to Your Existing React Project

Copy the client/src/ folder contents to your project
Install all dependencies from package.json
Update your routing/app setup as needed
